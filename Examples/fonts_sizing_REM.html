Font sizing with root-em (REM)

REM is relative to the font-size of the root element.

In this example root font-size has been set to 20px for the entire page using the code below.

<style>
:root { font-size: 20px; }
</style>

So, 1rem = 20px, 1.5rem = 30px, 2rem = 40px, 3rem = 60px, and so on.
