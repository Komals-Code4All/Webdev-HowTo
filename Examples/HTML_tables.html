HTML Table Basics

The follow tags are used to define an HTML table.

Tag Purpose
<table>...</table> start / end of table definition
<caption>...</caption> table caption
<thead>...</thead> headings sections
<th>...</th> table column heading
<tbody>...</tbody> table body section
<tr>...</tr> table row
<td>...</td> table data
<tfoot>...</tfoot> footer section


How to structure the HTML table tags

An HTML table consists a head, a body and a footer section.

Each section can contain one or more table rows (<tr>), and each row can contain one or more table data (<td>) items.

Table Examples

There are attributes that can be set when defining tables, for example 'rowspan' and 'colspan' can be used to span across multiple cells.

When there are complex data sets to put into a table it is often best to use something like JavaScript to build the table, and then have it displayed on the web page.

The code below is a nice way to style every other row, to make the table easier to read.

tr:nth-child(odd) { background-color: lightgreen;} or tr:nth-child(even) { background-color: lightgreen;}
