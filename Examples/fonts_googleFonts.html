Using Google Fonts https://fonts.google.com

In HTML link to Google's stylesheet that contains the desired font family 


<link rel="stylesheet" 
  href="https://fonts.googleapis.com/css?family=Tangerine "	/>

and then style the elements directly or by defining a CSS class


.h1 {
  font-family: "Tangerine", serif;
  font-size: 75px;
  text-shadow: 4px 4px 4px #aaa;
}
