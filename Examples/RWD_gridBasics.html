RWD using GRID System.

NB: This is not strictly RWD. The columns do NOT stack on top of each other for mobile phone screens. See other RWD examples provided here.

For the GRID System, imagine the page is divided into 12 columns.

Using 100% of the page's width means each column is 100% / 12 = 8.33333% wide. In CSS define classes for each column, where col-01 is 8.33% wide, col-02 is 16.66% wide, and so on.

Now, one 'row' is an outer container made up of these twelve columns. Divide the row into as many columns as you need, but remember that the total width of all columns must = 12 columns.
