Gradient Fill Backgrounds

You can apply CSS {background-image : gradient-fills} at the <body> level to fill the whole page.

Here, CSS classes are defined and used to fill the four <div>s below.

{background-image: linear-gradient(red, yellow);}

Linear fill, defaults top to bottom


{background-image: linear-gradient(to right, red, yellow);}

Linear fill, set to go left to right


{background-image: linear-gradient(to left bottom, red, yellow);}

Linear fill, set to go top right to bottom left corner


{background-image: linear-gradient(to right bottom, red, yellow);}

Linear fill, set to go top left to bottom right corner
