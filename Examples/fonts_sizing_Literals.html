Font Sizing with Literal Keywords

Fonts can be sized using percentages, pixels or these literal keywords;

  • font-size: x-small Sets the font-size to an extra small size
  • font-size: small Sets the font-size to a small size
  • font-size: large Sets the font-size to a large size
  • font-size: x-large Sets the font-size to an extra large size
