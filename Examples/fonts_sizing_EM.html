Font Sizing with EM

By default font size is 16px, or 1em.

"EM" represents upper-case letter M's width. Other letters are then sized relative to this width.

<style>

.smaller { font-size: 0.75em; }

.larger { font-size: 1.25em; }

</style>


Embed <div class=smaller> to get fonts getting relatively smaller by 0.75em
